Quick Guide


Before Starting

Make shure you have the server software installed on the computer you want to remote control. You can download the software at: Qt-Apps.org


Connecting

- Connect your smart phone and your computer to a wireless network.

- Start the server software on your computer.

- Click "Search Servers" on you smart phone to discover the servers on your network.

- Sometimes the client can not find the server automatically, you have to type in the IP-address manually in the advanced tab.

- Click on the server you want to use.

- Start using QRemoteControl.

- If the connection does not work, try to open port 5487 for UDP and TCP protocol on your firewall.
